# RU-LineArt Agent Quickstart

This folder ships with machine-readable API discovery files.

- Manifest: `tool_manifest.json`
- OpenAPI: `http://127.0.0.1:8000/openapi.json`
- Conversion endpoint: `POST http://127.0.0.1:8000/convert`

## Start local API

```bash
.venv/bin/sketch2rhino serve --host 127.0.0.1 --port 8000
```

## Convert request

```json
{
  "image_path": "/absolute/path/to/input.png",
  "output_path": "/absolute/path/to/output.3dm",
  "config_path": "/absolute/path/to/config.yaml",
  "debug_dir": "/absolute/path/to/debug",
  "include_report": true
}
```
